# Anticapture API

Programmatic access to [Anticapture](https://app.anticapture.com) — DAO
governance, delegation, treasury, and voting analytics indexed from onchain and
offchain sources, for protocols like ENS, Uniswap, Optimism, and Arbitrum.

There are three ways to consume the platform:

<div className="ac-cards">
  <Card
    to="/api-reference/anticapture-gateful-rest-api"
    title="REST API"
    badge="HTTP"
  >
    Query governance data over plain REST — proposals, delegations, voting
    power, treasury, token metrics, and more, scoped per DAO.
  </Card>
  <Card to="/mcp" title="MCP server" badge="Agents">
    Plug Anticapture into Claude, Cursor, or any Model Context Protocol client
    and query DAO analytics in natural language.
  </Card>
  <Card to="/webhooks" title="Webhooks" badge="Push">
    Register an HTTPS endpoint and get signed notifications pushed to you,
    instead of polling the API.
  </Card>
</div>
